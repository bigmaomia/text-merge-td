# 「赵云与阿斗」文字合成塔防 — UI 设计规范 v1.0

> 文档归属：UI 设计组（小D派发 / 飞飞文档对齐）
> 设计稿基准尺寸：**750 × 1334**（微信小游戏标准设计稿，@2x）
> 交付对象：思思（编码开发）
> 风格基调：三国题材 · 像素/Q版混搭 · 暖色战旗 · 低龄化但不幼稚

---

## 一、设计风格定位

### 1.1 视觉调性
- **核心感受**：热血 + 轻松 + 策略感。用三国战旗的厚重度做"底"，用 Q 版像素圆角做"面"，二者对冲出"老少皆宜"的休闲策略感。
- **气质关键词**：烽火、旌旗、黄沙、合成光、羁绊。
- **差异化**：不走高冷写实（避免审核/成本风险），也不走纯萌系（避免失去三国味）。采用 **"像素骨架 + Q版面皮"** —— 图标/插画为像素笔触，UI 容器为圆角厚描边面板（类卡牌手游）。

### 1.2 色彩体系（HEX）

| 角色 | 名称 | HEX | 用途 |
|------|------|-----|------|
| 主色 | 战旗红 | `#C0392B` | 主按钮、强调、血条、重要边框 |
| 主色辅 | 暮金 | `#E0A82E` | 星级、勋章、稀有度高亮、标题描边 |
| 辅色1 | 墨青 | `#2C3E50` | 面板底色、深色背景、文字底 |
| 辅色2 | 竹青 | `#3E8E7E` | 技能/安全态、羁绊激活、成功提示 |
| 辅色3 | 烽烟紫 | `#6C5B7B` | 武将专属、法术技能、稀有武将底 |
| 中性-深 | 玄夜 | `#1A1A24` | 战场背景、遮罩 |
| 中性-中 | 灰岩 | `#8A8A99` | 禁用态、次要文字、分隔线 |
| 中性-浅 | 宣纸 | `#F5EFE0` | 卡片面、文字底、弹窗背景 |
| 警示 | 警报橙 | `#E67E22` | 阿斗受击、低血量警告 |
| 危险 | 败血红 | `#E74C3C` | 失败、血量危急、扣血飘字 |

> **配色纪律**：单屏主色占比 ≤40%，宣纸/墨青做大面积底，战旗红与暮金只做"点睛"。

### 1.3 字体方案
- **标题字体**：`站酷快乐体` / 备选 `方正像素14`（像素描边感，用于主界面大标题、星级、爆字）
- **正文字体**：`思源黑体 CN` / `PingFang SC`（清晰可读，用于数值、说明、列表）
- **数字字体**：`DIN / Orbitron`（金币、波次、CD 倒计时用等宽数字，方便对齐跳动）
- **字号阶梯（设计稿 px，@2x）**：

| 层级 | 字号 | 字重 | 行高 | 用途 |
|------|------|------|------|------|
| 巨标题 | 56 | 900 | 64 | 主界面游戏名、武将登场名 |
| 大标题 | 40 | 800 | 48 | 界面标题、章节名 |
| 中标题 | 32 | 700 | 40 | 区块标题、按钮文字 |
| 正文 | 28 | 500 | 38 | 列表项、说明文字 |
| 小字 | 24 | 500 | 32 | 辅助说明、标签 |
| 微字 | 20 | 400 | 28 | 角标、CD数字、单位 |

### 1.4 图标 / 插画风格指南
- **像素图标**：16×16 / 24×24 像素网格对齐，2px 硬描边（描边色 `#1A1A24`），扁平填色，无渐变或单方向微渐变。
- **武将插画**：Q 版立绘（头身比 1:1.5），像素笔触上色，武器/铠甲做高光像素点；SSR/UR 加描金边 + 光晕。
- **兵种文字图标**：单字居中方块，Lv1-3 灰底、Lv4-6 青底、Lv7-9 紫底、Lv10 金底描红边；字用 `站酷快乐体` 加描边。
- **统一圆角**：UI 面板圆角 16px，按钮圆角 12px，卡片圆角 12px，小标签圆角 8px。
- **投影规则**：容器统一 `0 4px 0 rgba(0,0,0,.25)` 硬投影（像素感），不用模糊大投影。

## 二、界面布局设计（750×1334 基准）

> 以下坐标均为设计稿 px。安全区：左右 padding 24px，顶部状态栏高 88px，底部 Banner 高 100px。

### 2.1 主界面（Home）
- **顶部栏**（0–88px）：左「游戏名 赵云与阿斗（28px 描金）」；右「💎 120（24px 数字）」+ 头像入口。
- **战旗背景区**（88–760px）：动态三国城池/攻城场景，底部 30% 压暗渐变保证按钮可读。
- **四大主按钮**（横向排列，y≈480–620，单按钮 150×150）：
  - 开始战役（战旗红，🔥图标）、武将养成（墨青，🛡）、商店（暮金，🛒）、图鉴（竹青，📖）。
  - 间距 24px，居中排布，按钮下方 24px 小字标签。
- **底部功能栏**（y≈900–980，高 80，4 等分）：关卡选择 / 活动 / 排行 / 设置（图标+小字，灰岩底白字）。
- **Banner 广告位**（y≈1234–1334，高 100）：矩形占位 `#D9D2C2` 底 + 「广告」角标（左上 20px 灰字）。

### 2.2 战斗界面（Battle）— 核心
- **顶部状态栏**（0–88px）：左「波次 5/20」+ 暂停⏸（右上）；中「💰2350 💎120」；右「阿斗❤×10」缩略。
- **文字掉落区**（88–180px，高 92）：横条，露出 4–6 个待落文字方块（64×64，间隔 16），可合成时方块描金发光。
- **战场网格**（180–860px，高 680）：
  - 5 行 × 9 列格子，单格约 72×72，gap 6，整体居中（x=24 起，宽 702）。
  - 格子底：`#2C3E50` 半透明 + 1px 描边；空格 `#1A1A24` 40% 透明。
  - 敌军行进路线：从左上进入，蛇形向右下阿斗（右下角格）行进，路线用虚线金 `#E0A82E` 60% 标出。
  - 阿斗格：右下角（第5行第9列），显示阿斗 Q 版头像 + 血条（红底绿/青填充）。
- **技能栏**（860–960px，高 100，左下）：2–3 个圆形技能按钮（直径 84，间隔 24），中心图标 + 右下 CD 环形遮罩 + 数字。解锁前灰锁。
- **状态条**（860–960px，右下或顶部合并）：波次 / 金币 / 钻石 三件套。
- **激励视频浮动按钮**：右侧居中浮动（y≈520，宽 96 高 96，暮金圆），文案「看广告+3字」。

### 2.3 结算界面（Result）
- **遮罩**：半透明玄夜 `rgba(26,26,36,.82)`，居中弹窗（宽 630，高 760，圆角 24，宣纸底）。
- **星级评价**：顶部 3 颗星（直径 80，间隔 24），未得星灰 `#8A8A99`，得星金 `#E0A82E` + 弹跳动效。
- **结果标题**：胜利「长坂坡大捷！」/ 失败「阿斗倒下了…」巨标题。
- **奖励掉落**：3–4 个奖励格（金币/钻石/碎片/经验），图标+数量飘入。
- **按钮组**（底部）：重试（灰岩描边）/ 看广告双倍（战旗红，仅胜利有）/ 下一关（暮金，主按钮）/ 返回（文字按钮）。

### 2.4 武将养成界面（Hero）
- **左栏 武将列表**（宽 200，y 88–1234）：竖排头像卡（80×80 + 名 + 星级），选中态战旗红左边框 6px。
- **右栏 武将详情**（x 224 起，宽 502）：
  - 顶部：武将大立绘（200×200）+ 名（巨标题）+ 稀有度徽章（SR/SSR/UR 色块）。
  - 属性条：等级 Lv.12、战力 12345（数字字体）。
  - 养成四宫格：升星 / 装备 / 技能 / 羁绊（图标+进度）。
  - 底部操作：升级（金币）、升星（碎片）、装备槽（3）。

### 2.5 商店界面（Shop）
- **顶部**：Tab 切换（热卖 / 钻石 / 碎片 / 礼包），选中态底部战旗红下划线。
- **商品列表**（单列卡片，宽 702，高 160，间隔 20）：
  - 左：商品图标（120×120）；中：名+描述+原价/现价（战旗红现价）；右：购买按钮（暮金圆角）。
  - 月卡/礼包卡加「超值」飘带（战旗红斜角）。
- **特惠位**：顶部横幅大卡（702×200）放新手礼包/月卡，带倒计时。

## 三、交互反馈规范

| 交互 | 视觉反馈 | 动效参数 | 听觉 |
|------|----------|----------|------|
| 文字配对成功 | 闪光 + 方块跳动 + "+1" 飘字 | 缩放 1→1.2→1，120ms；飘字上浮 40px 渐隐 400ms | 合成"叮" |
| 合成高级兵种 | 全屏边缘金闪 + 兵种图标放大定格 | 闪光 200ms，图标 pop 1→1.4→1 240ms | 号角 |
| 武将登场 | 立绘从下弹入 + 名字横幅扫光 | 弹入 300ms 缓出，横幅 400ms 扫光 | 登场台词 |
| 敌军攻击阿斗 | 画面震动 + 阿斗红闪 + 血条掉血 | 抖动 ±6px 200ms，红叠层 30% 150ms | 警报 |
| 技能释放 | 全屏技能专属特效 + 按钮缩放回弹 | 特效 600ms，按钮 0.9 回弹 150ms | 技能音 |
| 三星通关 | 金色粒子 + 三星依次弹跳 + 光柱 | 星 0/120/240ms 依次弹，光柱 500ms | 胜利乐 |
| 关卡失败 | 灰滤镜 + "阿斗倒下"缓降 + 重试震动 | 灰阶 400ms，倒下 500ms | 失败音 |

> **通用节奏**：所有反馈总时长 ≤ 700ms，避免阻塞操作；合成/攻击类用"快进快出"，登场/结算类用"慢起慢落"。

## 四、设计参数标准（速查）

- **安全边距**：左右 24，顶 88（含状态栏），底 100（Banner）。
- **网格系统**：8px 基准（间距 8/16/24/32/48），战斗格 72 为 8 的倍数。
- **按钮尺寸**：主按钮高 96（战斗技能 84 圆），次按钮高 72，文字按钮高 56。
- **圆角**：面板 16 / 按钮 12 / 卡片 12 / 标签 8。
- **描边**：常规 2px `#1A1A24`，强调 3px 战旗红，稀有 3px 暮金。
- **硬投影**：`0 4px 0 rgba(0,0,0,.25)`（像素风统一），禁用模糊阴影。
- **CD 环**：技能按钮内 84 直径环，弧长 = 剩余/CD，色 玄夜 70% 遮罩 + 白数字。
- **血条**：高 16，圆角 8，底 `#1A1A24`，填充按类型（阿斗青/敌红）。
- **字号**：见 1.3 阶梯；数字统一等宽字体保障跳动对齐。
- **图标输出**：SVG/PNG 两倍图（@2x / @3x），像素图禁抗锯齿放大。
